Exercice 4.7h Mesure de l'énergie

Qu'indique la constante c sur un compteur. Sélectionnez les affirmations justes ci-après.

Pour les compteurs mécaniques, la constante c donne le nombre de tours effectués par le disque d'un compteur pour 1 kWh.
Pour les compteurs mécaniques, la constante c donne le nombre de kWh par tour du disque du compteur.
Pour les compteurs électroniques, la constante c donne le nombre d'impulsions (diode clignotante) par kWh.
Pour les compteurs électroniques, la constante c donne le nombre de kWh par impulsion (diode clignotante).
Voir section 4.7

Pour calculer la puissance absorbée par un récepteur, il est nécessaire de connaître la constante du compteur c, c’est-à-dire le nombre de tours qu’effectue le disque du compteur pour 1 kWh. Cette constante c est inscrite sur la plaquette signalétique du compteur.

Pour les compteurs électroniques avec affichage numérique, la constante de temps c est donnée en nombre d’impulsions par kWh, p. ex.: c = 1000 imp / kWh et n représente le nombre d’impulsions pendant le temps t (diode clignotante).
Les affirmations justes sont :
- Pour les compteurs mécaniques, la constante c donne le nombre de tours effectués par le disque d'un compteur pour 1 kWh.
- Pour les compteurs électroniques, la constante c donne le nombre d'impulsions (diode clignotante) par kWh.
